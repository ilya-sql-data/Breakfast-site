# Breakfast-site

Сайт для оформления заказа завтрака с отправкой уведомления в Telegram-бота.

> Примечание: этот проект был изначально разработан в период с апреля 2025 по июль 2025, еще до создания этого GitHub-аккаунта 31 января 2026 года. Позже я перенес его сюда, когда заново оформлял свое портфолио и перестал использовать старый аккаунт.

## О проекте

Breakfast-site — это pet-проект, в котором пользователь может выбрать позиции завтрака через веб-интерфейс, оформить заказ и получить результат с сохранением данных в базе.

После отправки формы заказ:
- сохраняется в SQLite-базу данных;
- может быть просмотрен в общем списке заказов;
- отправляется в Telegram в виде уведомления.

Проект был сделан как учебная практика по работе с backend-логикой, HTML-шаблонами, формами, базой данных и интеграцией со сторонним API.

## Основной функционал

- главная страница сайта;
- страница оформления заказа;
- выбор напитка, основного блюда и фрукта;
- проверка обязательных полей формы;
- сохранение заказа в SQLite;
- просмотр списка заказов;
- сортировка и фильтрация заказов;
- отправка уведомления о новом заказе в Telegram.
